## Overview
A web application that shows the Air Quality Index (AQI) for selected cities. It helps users check pollution levels easily.

## Features
- User registration, login, and logout system  
- User input validation (email, password, etc.)  
- Option to change background color  
- Select preferred cities  
- AQI is shown only for the selected cities  
- Simple and user-friendly interface

## How to Run
1. Install XAMPP  
2. Move the project folder into `htdocs`  
3. Import the database file into phpMyAdmin  
4. Start Apache and MySQL  
5. Open `http://localhost/AQI` in your browser
